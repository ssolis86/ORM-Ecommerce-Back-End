# ORM-Ecommerce-Back-End


## Description

https://git.heroku.com/orm-ecommerce-back-end.git is a web application that focuses on back-end management of an ecommerce's business inventory.

## Installation
- npm install "dotenv"
- npm install "express"
- npm install "mysql2"
- npm install "sequelize"
- complete .env file
- run "npm run watch"
- run "npm start"
